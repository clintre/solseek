Liste les paquets (Tous ou Installés)

Vous permet de rechercher et gérer les paquets disponibles dans le 
dépot utilisé.

Lorsque la liste est affichée, les options suivantes sont disponibles :

    - Rechercher un paquet dans la liste
    - Afficher les informations d'un paquet
    - Opérations sur les paquets, notamment Installer, Réinstaller, et Supprimer

Guide
    - Lister Tout : montre tous les paquets du dépot
    - Lister les paquets installés : montre seulement les paquets installés
    - Lister les paquets installés par l'utilisateur : montre uniquement les paquets installés par l'utilisateur
    - Lister tous les Flatpaks : Montre tous les Flatpaks présents sur flathub
    - Lister les Flatpaks installés : Montre seulement les Flatpaks installés
    - Commencer à taper pour rechercher dans la liste
    - Utiliser les flèches haut/bas sur le clavier ou faire défiler avec la souris/pavé tactile
    - Pour sélectionner un paquet, appuyer sur [Entrée] ou double-cliquer avec la souris

Notes
    - Paquets = Paquets système / EOPKG
    - Tous = Paquets installés et disponibles
    - Si le Flatpak n'est pas activé, il n'apparaîtra pas dans la liste
