Packages & Flatpaks

List, search, and manage packages.

When viewing the list, the following options are available.

    - Searching the list for a package
    - Viewing package information
    - Actions for packages, including Install, Reinstall, and Remove

Guide
    - Packages - All: All system packages from the repo
    - Packages - Installed: All installed system packages
    - Packages - User Installed: System packages installed by user
    - Flatpaks - All: Shows all flatpaks from flathub
    - Flatpaks - Installed: Shows only installed flatpaks
    - Type to Search List
    - Up / Down on Keyboard or Scroll on Mouse/Touchpad
    - To select package [Enter] on kb or double click w/ mouse

Notes
    - Packages = System Packages / EOPKG
    - All = Both installed and available
    - If Flatpak is not enabled, it will not show in the list
