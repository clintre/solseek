List Packages (All or Installed)

Allows you to search for and manage packages available in the
enabled repository.

When viewing the list, the following options are available.

    - Searching the list for a package
    - Viewing package information
    - Actions for packages, including Install, Reinstall, and Remove

Guide
    - List All: Shows all packages from the repo
    - List Installed: Shows only installed packages
    - List User Installed: Shows only packages installed by the user
    - Type to Search List
    - Up / Down on Keyboard or Scroll on Mouse/Touchpad
    - To select package [Enter] on kb or double click w/ mouse
